# 2026.08 韓國西部之旅

本 repository 存放 2026.08 韓國西部之旅的手機版 Naver Map 導航網站。

網站首頁為 `index.html`，可直接用瀏覽器開啟。主要用途是在旅途中快速依日期與類型篩選住宿、景點、美食地點，並開啟 Naver Map 導航。

## 資料處理原則

原始規劃書包含護照號、身分證字號、生日等個人敏感資料。製作公開或半公開旅遊資訊網站時，不應直接放入這些個資。

網站可優先使用行程、交通、住宿、景點、美食、地圖連結、注意事項與詞彙翻譯等資訊。若需要處理參與者資料，應只保留必要的團別或暱稱，並避免公開完整證件資訊。
