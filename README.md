# 2026.08 韓國西部之旅

本 repository 存放 2026.08 韓國西部之旅的手機版旅程資訊網站。

網站首頁為 `index.html`，可直接用瀏覽器開啟。進入後可選擇兩種模式：

- 旅程總覽：給一般參與者快速理解每天行程，包含每日主題、地點介紹、參考連結與圖片。
- 快速導航：延續原本設計，依日期與類型篩選住宿、景點、美食地點，並開啟 Naver Map 與停車搜尋。

## 資料處理原則

原始規劃書包含護照號、身分證字號、生日等個人敏感資料。製作公開或半公開旅遊資訊網站時，不應直接放入這些個資。

網站可優先使用行程、交通、住宿、景點、美食、地圖連結、注意事項與詞彙翻譯等資訊。若需要處理參與者資料，應只保留必要的團別或暱稱，並避免公開完整證件資訊。

`story-data.js` 與 `assets/trip/` 是由 HTML 匯出檔擷取出的公開行程內容，包含旅程總覽使用的地點介紹、參考連結與壓縮圖片。原始 ZIP 與規劃資料不應直接發布到 GitHub Pages。

首頁另提供 Google Drive 連結作為完整行程 PDF 下載入口。

## 圖片來源

旅程總覽每日主圖仍使用外部代表圖。各地點介紹圖片則由使用者提供的 HTML 匯出檔轉成 `assets/trip/` 內的壓縮 JPG，供手機瀏覽時載入。
